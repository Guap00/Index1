Datakommunikation & Webbadresser

1 ) HTTP - hypertext transfer Protocol - är ett exempel på ett protokoll som beskriver vilket regelverk eller hur kommunikationen mellan olika enheter ska fungera. HTTP sköter kommunikationen mellan webbläsaren och webbservern. Det protokollet innehåller fler refernser vilket då begär fler filöverföringar att webbläsaren måste begära fler filer.

HTTP, TCP och IP är alla olika överföringstekniker som används på nätet. HTTP syftar på kommunikationen på programnivån medan TCP (Transmission Control Protocol) och IP (Internet Protocol) är på en lägre nivå mot det fysiska nätverket. TCP gör om information och organiserar paket så att IP kan adressera - med en IP-adress mot en router på nätet som sen skickar adresserade paket vidare till den väg som passar bäst. IP och TCP samverkar med varandra, sänder och tar emot och meddelar om paket kommer fram eller ej. Därefter gör TCP om paketet igen till den ursprungliga informationen eller meddelandet.
